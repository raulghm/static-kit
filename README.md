# Starter web kit
> for static web development

[http://github.com/raulghm/static-kit](http://github.com/raulghm/static-kit)

---

## Features

* Gulpjs for automation tasks
* Bower ready
* Assemble
* BrowserSync
* Livereload
* SASS
* Autoprefixer
* Sourcemaps
* Minify
* Imagemin
* HTMLmin
* CSSmin
* UglifyJS
* Notify

## How to use

### Serve

```
$ gulp 
```

### Build

```
$ gulp build 
```

### Distribution

```
$ gulp build --dist
```
